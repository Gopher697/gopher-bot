# gopher-workbench

A local-only, conservative MCP server that gives AI coding assistants access to:

- Personal project SOP resources.
- Explicitly configured project summaries and notes.
- Read-only git status and diff.
- Named, preapproved commands.
- Timestamped session-note appends inside configured project folders.

The server intentionally has no general shell execution tool and does not scan the
home directory for projects.
